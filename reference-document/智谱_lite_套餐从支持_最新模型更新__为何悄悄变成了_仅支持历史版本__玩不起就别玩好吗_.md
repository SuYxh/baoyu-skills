可以看到，这不是一次普通的服务调整，这绝对是一次实打实的**核心权益的实质性缩水**。

我试图找到智谱官方关于"Lite套餐模型支持变更"的明确公告，但搜索结果显示：

*   • **2026年1月21日**：智谱因算力紧张宣布限售，每日销售量降至20%
    
*   • **2026年2月10-12日**：官方文档更新，明确"所有套餐均支持GLM-4.7及历史文本模型"，但"仅Max套餐新增支持GLM-5" bigmodel\[1\]
    
*   • **2月12日前的订阅者**：官方承诺"在您当前订阅套餐的有效期内，套餐的用量额度不变" docs.bigmodel\[2\]
    

注意这里的措辞：智谱只承诺了"用量额度不变"，**并未明确承诺"模型支持范围不变"** 。 docs.bigmodel\[3\]

这就是问题的核心所在。

### 协议的"变"与"不变"

我仔细阅读了智谱的《订阅服务协议》，发现了这样的条款：

> "智谱有权根据业务需要调整订阅服务的内容和形式，并在合理范围内提前通知您" docs.bigmodel\[4\]
> 
> "智谱有权根据需要修改本协议，并在平台上发布，修改后的协议自发布之日起生效" docs.bigmodel\[5\]

![](https://mmbiz.qpic.cn/mmbiz_png/sc9vl5waw2oiazcTz1xYF4f0sewa94voXS6aCw3gKwjhX4lLeuxr6LY4zicxATlAUHmqJDEF4aoL7EBbKEGkNKlXTxq6iaYTN0cw1DdYJYrRak/640?wx_fmt=png&from=appmsg)

但是，你通知我了吗，我从没看到过邮件，或者短信，或者电话呀？

这类格式条款在互联网服务中极为常见，几乎每家公司都有。但**常见不代表合理，更不代表在任何情况下都有效**。

### 用户买的到底是什么？

回到最初的截图 ，用户购买时页面清楚地写着：

*   • "订阅期内，享受同等级最新旗舰模型更新"
    

这句话构成了**用户购买决策的核心依据**。换句话说，用户支付192元年费，买的不仅是当下的GLM-4.7，更是对未来一整年"模型持续升级"的预期。

而现在，智谱用一个格式条款，把这个承诺变成了"历史模型"

### 算力紧张是事实

从多方信源来看，智谱确实遇到了严重的算力瓶颈：

*   • 1月下旬开始限售，每日销售量降至20%
    
*   • 高峰期（工作日15:00-18:00）出现并发限流、响应变慢
    
*   • 官方公告承认"用户量激增，算力资源阶段性紧张"
    

这些信息是公开的、可验证的。作为一家国产AI公司，面对算力约束是客观现实。

### 但这能成为单方面改协议的理由吗？

**不能。**

算力不足有很多解决方案：

1.  1. **限制新用户购买**（智谱已经做了）
    
2.  2. **对存量用户限流降速**（在高峰期已经发生）
    
3.  3. **涨价**（2月12日确实涨了30%）
    
4.  4. **允许退款**
    

但**最不应该做的，就是单方面降低服务标准，还不给已购用户选择权**。

用户在购买时，"最新模型支持"是明确承诺的权益。如果智谱现在无法兑现，合理的做法是：

*   • 公开说明情况
    
*   • 给予补偿或退款选择
    
*   • 至少对已购用户提供过渡期
    

而不是悄悄修改页面描述，然后用一句"我们有权修改协议"来搪塞 。

### 格式条款的边界：法律不会站在"霸王条款"一边

#### "有权修改协议"≠ 可以随意修改

智谱的订阅协议中确实写明了修改权 ，但这类格式条款在法律上是有限制的： zhipu-ef7018ed.mintlify\[6\]

根据《消费者权益保护法》和《民法典》：

*   • **显失公平的格式条款无效**
    
*   • **排除消费者主要权利的条款无效**
    
*   • **经营者不得以格式条款单方面免除自己的责任、加重消费者责任**
    

"订阅期内享受最新模型"是用户付费的核心诉求，这不是可有可无的附加服务，而是**主要权利** 。智谱用格式条款单方面取消这个权利，在法律上站不住脚。

我找到了一个2025年12月的案例：一位用户因GLM编码套餐效果不佳申请退款被拒，通过**市场监督管理局介入**，智谱最终同意退款 。

### 这不仅是192元的问题，更是诚信的问题

因为这不是钱的问题——192元对大多数开发者来说不是巨额损失。真正让人愤怒的是**被背叛的感觉**。

你在9月或10月购买套餐时，看到的是"享受最新模型更新"的承诺 。你基于这个承诺支付了年费，放弃了其他选择（比如直接用Claude、或买其他家的服务）。

结果4个月后，你发现这个承诺变成了一句空话 。  
这不是服务降级，这是**欺诈感**。

### 智谱的短视，还能拿什么去信任

我无法理解智谱为何要做这个决定。

即使从纯商业角度考虑：

*   • **品牌损失**：这件事一旦发酵，对智谱的信誉是致命打击
    
*   • **法律风险**：已购用户完全可以集体投诉，甚至提起诉讼
    
*   • **用户流失**：谁还敢买年度套餐？明年续费率会怎样？
    

相比之下，给已购Lite用户继续提供新模型支持，能花多少算力？这些是你的种子用户、早期支持者，是最应该维护的群体。

为了省这点成本，毁掉整个品牌的信任基础，这笔账怎么算都不划算。

### 不能接受：契约精神不能用"算力不足"来解释

我不是在苛责智谱面临的算力困境。国产AI公司的生存压力，我们都看在眼里。

但**困难不是理由，契约就是契约**。

你可以涨价，可以限售，可以对服务进行合理调整。但你不能：

*   • 在用户付费后单方面降低核心服务标准
    
*   • 用格式条款来规避自己的责任
    
*   • 对已购用户连个正式说明都没有
    

这不是"国产模型算力不足"的问题，这是**基本商业伦理**的问题。**用户的信任不能这样被辜负**。
